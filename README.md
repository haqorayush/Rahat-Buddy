## üìù About the Project

**Rahat-Buddy** ("Rahat" means "Relief" in Hindi/Urdu) is a chatbot initiative aimed at bridging the information gap during emergencies. Whether it's floods, earthquakes, or other natural calamities, Rahat-Buddy aims to serve as a first point of contact for users seeking immediate guidance, safety protocols, or emergency helpline numbers.

## ‚ú® Key Features

* **Emergency Information**: Provides quick access to critical helper numbers and contact details for relief agencies.
* **Safety Protocols**: Offers step-by-step guides on what to do before, during, and after various types of disasters (e.g., Earthquakes, Floods, Cyclones).
* **Interactive Chat Interface**: A user-friendly command-line or GUI interface (depending on implementation) to answer user queries effectively.
* **Lightweight**: Built with Python for easy deployment and usage.

## üöÄ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

* Python 3.8 or higher installed on your system.

### Installation

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/haqorayush/Rahat-Buddy.git](https://github.com/haqorayush/Rahat-Buddy.git)
    ```

2.  **Navigate to the project directory**
    ```bash
    cd Rahat-Buddy
    ```

3.  **Install Dependencies**
    * Since there is no `requirements.txt` yet, please check the top of `main.py` for imported libraries and install them using pip.
    * *Example (common for chatbots):*
        ```bash
        pip install requests
        # Add other libraries found in main.py here
        ```
